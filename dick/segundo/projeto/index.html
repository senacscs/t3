Problema

Como fazer o aluno com altas habilidades/superdotados se sentirem incluídos emocionalmente na sala de aula brasileira?

Objetivos


Principal

Desenvolver a plataforma para auxilio de pessoas diagnosticadas com AH/SD.


Específicos

• Popularizar e desesteriotipar os termos;

• Dar assistência e suporte, tanto emocional, quanto educacional para o aluno com a AH/SD;

• Diagnosticar o mais cedo possível esses alunos;

• Incentivar o acompanhamento psicológico fora da escola;

• Entender o aluno;

• Conscientizar os colegas;

• Fazer o aluno com AH/SD se sentir confortável e incluído emocionalmente na sala de aula brasileira;

Soluções


A solução seria a criação de uma plataforma, com acompanhamento online.

Acompanhamento


Será por meio de uma plataforma, liga a uma IA por meio de uma API, fazendo assim cada plano ser único para oferecer uma abordagem melhor para o aluno.
