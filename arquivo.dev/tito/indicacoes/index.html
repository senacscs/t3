Site de Conteúdos

Tabnews

O Tabnews teve seu lançamento recentemente, porém com uma comunidade muito animada com o projeto. Ele é código-aberto, significando que qualquer um pode participar do seu desenvolvimento.

Canal do Youtube

Fireship

O Youtube caminha junto com o front-end, o site possui criadores de conteúdo de diversos países que têm o assunto de programação front-end. O canal americano Fireship trata sobre programação front-end, back-end e muitas outras áreas.

Influenciador

Filipe Deschamps

A internet possui diversos influenciadores que tratam sobre programação, um dos que mais admiro é o Filipe Deschamps. Ele é o criador do site Tabnews e cria conteúdos sobre tecnologia no seu canal do Youtube.

Podcast

FalaDev

O Fala Dev é o podcast da Rocketseat, plataforma para o ensino de programação, que trata sobre diversas áreas da criação de programas, principalmente sobre arquitetura de software.

Tutorial sobre front-end

Começando com React

React é uma bilioteca para criação de UIs web. Ele usa o sistema de componentes para minimizar a reescrita de código e maximizar a produtividade. Esse tutorial, criado pela criadora do Firefox, ensina o passo a passo para iniciar com o React.
