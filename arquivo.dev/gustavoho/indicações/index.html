O que é borda no css

A border CSS, ou bordas aplicadas no estilo em cascatas, serve para que você
estilize seu site da melhor forma. Ela permite separar um espaço do layout de
uma página, definindo onde vai aparecer. Enquanto que a linguagem CSS permite
que isso seja feio de forma mais flexível.
