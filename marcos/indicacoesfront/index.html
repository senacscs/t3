Website de Conteúdos (documentação)

MOZILLA CSS DOCS

A documentação do MDS CSS, desenvolvido pela Mozilla para o Firefox, é uma valiosa fonte de informações para iniciantes no front-end. Ela fornece explicações detalhadas e exemplos práticos sobre como aplicar estilos CSS em páginas web, desde conceitos básicos até recursos avançados. Através da documentação, os iniciantes podem aprender como usar corretamente as propriedades de estilo, compreender a sintaxe do CSS, solucionar problemas de layout e entender as melhores práticas de desenvolvimento. A documentação é escrita de forma acessível e amigável, tornando-a uma ferramenta útil para ajudar os iniciantes a criar páginas web com aparência profissional e interativa utilizando o MDS CSS no Firefox.

Canal no Youtube de Conteúdo Front-End

TRAVERSY MEDIA

Traversy Media, liderado por Brad Traversy, é um canal popular no YouTube que oferece tutoriais gratuitos de desenvolvimento web, incluindo front-end, back-end e várias tecnologias relacionadas. Os vídeos de Traversy Media são conhecidos por serem claros, detalhados e bem explicados, tornando-os adequados para iniciantes e desenvolvedores experientes. O canal abrange uma ampla gama de tópicos front-end, como HTML, CSS, JavaScript, frameworks populares como React, Vue e Angular, e muito mais.
