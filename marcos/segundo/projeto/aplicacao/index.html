Aplicação técnica: HelpXPress

Como Funciona?

O HelpXPress é uma extensão do chrome, que permite a visualização de informações gerais do cliente. Integrada á um banco de de dados, essas informações permitem que o atendente tenha um melhor entendimento do cliente, e assim, possa oferecer um atendimento mais personalizado. A extensão busca na página o email do cliente de forma automática, e com base nesse email, busca as informações do cliente no banco de dados. Essa integração é específica para a plataforma de atendimento que a MK utiliza, o Movidesk. Portanto, funciona exclusivamente dentro do ambiente da MK, e não é possível utilizá-la em outros casos.

Tecnologias

A extensão foi desenvolvida utilizando HTML, CSS e JavaScript. O banco de dados foi desenvolvido utilizando PostgreSQL, e a API é em Golang. Por de baixo dos panos, existem diversas DAGs que realizam as buscas e os cálculos necessários para buscar as informações de diversas bases e consolidá-las em uma única base de dados. A API é hospedada em uma máquina na MK, bem como os servidores do banco de dados.

HTML, CSS, JS, PostgreSQL, Golang, Airflow, Docker, Git, Github Actions

O projeto proporcionou experiências técnicas e de gestão de projetos e pessoas. Atuei no desenvolvimento FullStack, planejamento e configuração de ambientes (DevOps). A aplicação está em faze de testes real com a equipe de atendimento da MK. A expectativa é que a aplicação seja aprimorada e após os testes, seja disponibilizada para toda a equipe de atendimento e utilizada no dia a dia.
