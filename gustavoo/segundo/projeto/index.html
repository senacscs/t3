Gerenciamento de validade de produtos em mercados de médio e pequeno porte.

Sobre o Projeto

O Projeto Prazo-zero tem como objetivo desenvolver um sistema de gerenciamento de validade de produtos em mercados de médio e pequeno porte. Utilizando um aplicativo que faz a leitura dos códigos de barras e dos lotes dos produtos, o sistema alerta sobre os produtos próximos do vencimento ou vencidos.

Metodologia

O projeto utilizou métodos de pesquisa em mercados locais para entender as necessidades dos proprietários e clientes. Ferramentas tecnológicas como leitura de códigos de barras foram implementadas para facilitar o controle de validade dos produtos.

Resultados Esperados

Os testes realizados mostraram que o aplicativo é capaz de identificar corretamente a validade dos produtos e sugerir ações para produtos próximos do vencimento, como promoções e doações, contribuindo para a redução do desperdício de alimentos.
